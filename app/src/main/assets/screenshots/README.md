## How To Capture

1. Open the real Rapido, Ola, or Uber app on a phone.
2. Go through the booking flow without completing a real ride.
3. Take screenshots for each step.
4. Hide personal details such as phone numbers, home address, driver name, vehicle number, exact location, and payment details.
5. Save each image as `.png` with the exact name above.

The app will automatically show these real screenshots. If a screenshot file is missing, it will show the simple guide picture instead.
